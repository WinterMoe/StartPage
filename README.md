# Start Page

This is a simple start page for myself to use to replace my new tab window. Its main features are the following:

- Keeps track of time/date based off local time of device
- Rotates through 10 images and different themes can be set and changed via the settings menu
- Has OpenWeatherMap API integration so weather can be viewed
- Searchbar with 7 options for search engine providers
- 9 Preset links with the ability to change them via the settings menu
- Calendar built in and can be viewed by clicking on date/time
